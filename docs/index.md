# 体系学習テキスト

このサイトは、**音声 (Audio)・強化学習 (Reinforcement Learning)・LLM・Physical AI** などの分野を、
基礎から体系的に積み上げて学ぶための技術テキストです。
すべての分野を**同じ構成・同じ品質**で書くことを重視しており、章のフォーマットは統一されています。

!!! tip "このテキストの方針"

    - **わかりやすさ最優先。** まず直感をつかみ、その後で理論・数式・実装へ降りていく。
    - **一から積み上げる。** 各分野はロードマップに沿って、前提知識を明示しながら順に学ぶ。
    - **手を動かす。** すべての章に実装（Python 中心）と演習問題が付く。
    - **日本語ベース＋英語用語。** 専門用語（gradient, attention など）は原語のまま使い、原論文・英語資料と接続しやすくする。

## 各章の構成

どの分野のどの章も、次の流れで書かれています。これにより、分野が変わっても同じ感覚で読み進められます。

```mermaid
flowchart LR
  A[学習目標] --> B[前提知識]
  B --> C[直感]
  C --> D[理論]
  D --> E[数式の導出]
  E --> F[実装]
  F --> G[演習]
  G --> H[まとめ]
  H --> I[参考文献]
```

| セクション | ねらい |
| --- | --- |
| 学習目標 | この章を読み終えたら何ができるようになるか |
| 前提知識 | 事前に知っておくべきこと（既習章へのリンク付き） |
| 直感 | なぜこれを学ぶのか・何の問題を解くのか |
| 理論 | 定義・定理・概念の正確な記述 |
| 数式の導出 | 結果を天下りにせず、ステップを追って導く |
| 実装 | 動かせるコード（Python 中心）で理論を確かめる |
| 演習 | 手を動かして定着させる（解答は折りたたみ） |
| まとめ | 要点の再確認と「次に学ぶこと」 |
| 参考文献 | 原論文・教科書・信頼できる資料 |
